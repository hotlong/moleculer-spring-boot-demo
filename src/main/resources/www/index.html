Moleculer Samples

  • Distributed, websocket-based drawing board

    Distributed drawing board where multiple users can draw. The drawing events are transmitted over the network, so the users can see each other's drawings.

  • WebSocket-based Chat Application

    The second example is a simple chat application. Like the first example, it uses WebSockets to transmit events.

  • Server Side Generated Image

    Molecular Services can not only produce JSONs. This example shows how to generate binary content using Moleculer Service. In this example, the generated binary content is a PNG image with a clock drawn on it.

  • Generating multilingual HTML with server-side template engine

    This example shows how HTML Template Engines can be used. The Moleculer Service generates a pageable HTML table that can be displayed in 45 languages.

  • File upload

    The example demonstrates that Moleculer services can not only receive requests in JSON format, but can also handle multipart content.

  • Media Streaming

    Moleculer Services can send media (eg. video or audio) files to the browser. When sending a video, the Moleculer Service handles partial requests, so the seeking function works in the Video Player window.

  • JMX Search

    A search interface to search the local JMX registry. The example shows how to use the JMX Service.

  • JMX Listener

    The example shows how to use event listeners. The service monitors the JMX registry and sends the changed data to the browser when its contents change. The data is sent to the HTML page via WebSocket.

  • Simple REST Service (math.add)

    This example illustrates the implementation of a simple REST call.

  • Simple REST Service (greeter.hello)

    This example also illustrates the implementation of a REST call.
